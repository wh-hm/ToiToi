# ToiToi

R.AとM.Hによる、4年生前期制作物。

## ToiToiについて

ToiToiとは、勉強アプリケーションです。友達と話せる「チャット」や勉強することがメモをとれる「Todoリスト」、わからないことや質問を記録できる「質問スペース」の三つが主な機能です。

## 開発の背景

勉強を始めようと思っても、一人ではなかなかやる気が出ない。
このアプリは、日々の学習内容をメモしたり、やるべきタスクを明確に整理したりすることで、学習のハードルを下げ、継続をサポートすることを目的に作成しました。

## 主な機能

### 1. 共有・招待機能

- **パーソナル＆グループ利用**: 自分一人の作業用としても、仲間との共同学習用としても切り替えが可能です。
- **招待システム**: 作成者が招待リンク（またはコード）を送ることで、特定の仲間とだけスペースを共有できます。

### 2. コミュニケーション（チャット）

- **リアルタイムチャット**: 勉強中の何気ないつぶやきや、励まし合いができます。
- **進捗報告**: 「今から◯◯の勉強始める！」といった合図を仲間に送れます。

### 3. Todoリスト

- **タスク管理**: やるべきことをリスト化し、進捗を可視化します。
- **ステータス管理**: 「未完了」「完了」をワンタップで切り替え。過去に頑張った「完了済みタスク」もいつでも振り返ることができ、達成感を味わえます。

### 4. 質問スペース（Q&A）

- **知恵の共有**: 勉強中に詰まったところを書き込み、共有しているメンバーからアドバイスをもらうことができます。
- **ナレッジ蓄積**: 過去の質問をストックしておくことで、自分専用の参考書代わりになります。

## 使い方

1. アプリを起動します。
2. その日の学習目標をタスクとして登録します。
3. 学習中に気づいたことやメモを記録します。
