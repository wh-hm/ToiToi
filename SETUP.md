# 開発環境構築手順書

このプロジェクトをローカル環境で動かすための準備手順です。

## 1. 事前準備（インストールが必要なもの）

以下のツールがインストールされているか確認してください。
開発時の各ツールのバージョンは以下の通りです。

| ツール         | バージョン | コマンド         |
| :------------- | :--------- | :--------------- |
| **Node.js**    | `v20.19.6` | `node -v`        |
| **npm**        | `10.8.2`   | `npm -v`         |
| **TypeScript** | `v5.9.2`   | `npx tsc -v`     |
| **ts-node**    | `v10.9.2`  | `npx ts-node -v` |

## 2. セットアップ手順

### 1. リポジトリの準備

#### GitHub Desktop を使う場合 (推奨)

1. GitHubDesktopを開き `File` > `Clone Repository` を選択。
2. Webサイトで対象のリポジトリを開き、`Code` > HTTPSタブに書かれているURLをコピー
3. 2でコピーしたURLを1の入力欄に貼り付ける
4. LocalPathを選択して、Cloneする。

### 2. セットアップ

1. ToiToiのディレクトリからVSCodeを選択してアプリを開く
2. ターミナルを立ち上げ、`npm install`と入力する
3. `npm run dev`でサーバー起動
4. 表示されるURLをクリックし、画面が開かれるか確認
